## FEW-SHOT STYLE EXAMPLE (English — match this structure; use the child's actual numbers)

**Question:** What is 1/2 + 1/3?

**Answer:**

👉 What is it:
1/2 + 1/3 = 5/6
That means five equal parts out of six parts of the same whole.

---

👉 How does it work:

1. We are adding 1/2 and 1/3.

2. We cannot add yet.
   1/2 means the whole is cut into 2 equal parts → each part is bigger.
   1/3 means the whole is cut into 3 equal parts → each part is smaller.
   So the pieces are not the same size.

3. To add them, we need the pieces to be the same size.
   So we need the same bottom number.

4. Let’s find a number that both 2 and 3 can make.
   Table of 2 → 2, 4, 6
   Table of 3 → 3, 6
   👉 6 comes in both tables, so we choose 6.

5. Now change 1/2 into sixths.
   First ask: 2 × what = 6? Answer: 3.
   So we multiply the **top and bottom by 3** (not by 6 — we use 3 because 2 × 3 = 6).
   Show it:
   2 × 3 = 6
   1 × 3 = 3
   👉 1/2 becomes 3/6
   (same amount, but now in smaller equal pieces)

6. Now change 1/3 into sixths.
   First ask: 3 × what = 6? Answer: 2.
   So we multiply the **top and bottom by 2** (because 3 × 2 = 6).
   Show it:
   3 × 2 = 6
   1 × 2 = 2
   👉 1/3 becomes 2/6

7. Now both fractions use sixths. Write the add like this:
   👉 3/6 + 2/6
   The bottom number is 6 for both, so the pieces match — now we can add.

8. Add the top numbers only. Keep the bottom 6:
   3 + 2 = 5
   👉 The answer is 5/6

---

👉 Example:
Cut a cake into 6 equal slices.

Half the cake = 3 slices
One third = 2 slices

Together = 5 slices
So that is 5/6 of the cake.
